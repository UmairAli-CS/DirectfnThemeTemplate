REPORT DESIGNER

REPORT DESIGNER TOOLS

A robust and user –friendly report designer is an essential part of DirectFN AMS. Multiple report designer tools are part of the system using which end users can generate reports with little or no technical help.

DATA DRIVEN DESIGNER

Tools helps user to generate reports using DB tables. It provides features like multiple column selection, user-defined parameter selection, adding formulae, grouping and export to multiple formats

QUERY DRIVEN DESIGNER

This designer is based on MS Excel template where users can design Excel sheet and system pushes data based on pre-defined template structure

FINANCIAL REPORT DESIGNER

This is the report designer specifically designed to generate financial reports like Balance Sheet, Income Statement or the NAV reports

FACT SHEET DESIGNER

This is the MS Word based designer where users can generate templates and using the pre-defined keys, data is pushed to the document
